# chat-with-stranger 채팅방 프로젝트

![cws1](https://user-images.githubusercontent.com/35681772/68169503-cae6e500-ffaf-11e9-871d-3c61802e30b0.gif)

## 사용된 기술
 * Web Socket
 * Spring & Spring boot
 * Spring JPA
 * MySQL
 * Javascript
 * Thymeleaf
 * JSON
 * HTML / CSS
 * Bootstrap
 

## 기능
 * 오픈채팅방
   * 일반 채팅
   * 공지 기능
   * 파일 업로드 기능 - 업로드 파일 사이즈 조정 필요
 * 랜덤채팅방(예정)
   * 비동기 처리로 랜덤한 상대와 채팅방 연결(예정)
